Java Class Template Generator
The Java Class Template Generator is a simple tool to slightly speed up the very repetitive process of creating classes in Java. Simply input the class attributes, then the constructor and default get and set methods will be generated.

Obviously, this does not fulfill the needs of every class, but is a good starting point when given many basic fields are required.
