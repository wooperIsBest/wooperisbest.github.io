Mandlebrot Set

Zoomed into the Set.

The Mandlebrot Set, rendered by the program in greyscale.
Lighter areas denote more iterations for that value.
This Mandlebrot Set renderer was one of my first graphical projects with C++ (Ironically, it does pretty much nothing that couldn't been done just as easily with C, but ah, well).

The actual coding aspect of this project is admittedly not too complex, but it was fun to work out the math behind the complex numbers.

The program can be controlled with W to zoom in and arrow keys to pan directionally.
