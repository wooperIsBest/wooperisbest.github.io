Input.py
Input.py is a small Python module I developed, mostly for my own sanity. When getting into larger Python projects, it was becoming increasingly annoying to have to write input validation each time user input was needed.

This module includes several functions: int_input, float_input, and choice. The first two repeatedly ask the user for an int or float, respectively, until a valid one is given, while the third asks for a string out of a certain list.

This also introduced me to importing my own Python modules into other programs.
