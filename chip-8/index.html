About
The Chip-8 Interpreter is designed to be able to read instructions written in the Chip-8 language. I personally only wrote the interpreter, and the ROM Instruction Sets (below) were created by others (credited individually). Select a ROM to begin, and use keys 0-9 and A-F depending on the ROM.

HEART MONITOR (by Matt) RPS (by SystemLogoff)
